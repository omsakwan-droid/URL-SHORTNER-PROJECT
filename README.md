# URL-SHORTNER-PROJECT
# 🔗 URL Shortener

A full-stack URL Shortener application built with Node.js, Express.js, PostgreSQL, Drizzle ORM, and React.

## 🚀 Features

- Shorten long URLs
- Redirect to original URLs
- REST API
- PostgreSQL Database
- Drizzle ORM
- Environment Variable Support
- Modern React Frontend

## 🛠 Tech Stack

### Frontend
- React
- Vite
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- PostgreSQL
- Drizzle ORM

## 📁 Project Structure

```
URL-SHORTNER-PROJECT
│
├── URL_SHORTNER_SERVICE
│
└── url-shortener-frontend
```

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/omsakwan-droid/URL-SHORTNER-PROJECT.git
```

Backend

```bash
cd URL_SHORTNER_SERVICE
npm install
npm run dev
```

Frontend

```bash
cd url-shortener-frontend
npm install
npm run dev
```
